To visualize the data copy in this folder param.h90, genview.sh and gen_xdmf.f90. You can take them form one of the examples:
 in this location from our of the examples. E.g.: 

cp ../../examples/abc_triperiodic/visu_ex/* .

Change param.h90 consistently with the input files (the .in files) and the number of fields you want to visualize.
